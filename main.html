Arsitektur yang
Elegan & Scalable.

Baileys Multi-Device

Koneksi stabil dengan library terbaru yang mendukung multi-sesi WhatsApp tanpa perlu QR scan berulang.

SQLite3 Persistence

Penyimpanan data yang ringan dengan WAL mode. Auto-save setiap 5 detik, data aman dan mudah di-backup.

Plugin-Based System

Tambahkan fitur baru hanya dengan membuat file di folder plugin. Hot-reload otomatis tanpa restart bot.

Worker Thread Architecture

Bot berjalan di worker thread terpisah. Crash otomatis di-restart tanpa downtime. CLI interface untuk monitoring real-time.

AI Rich Formatting

Respons AI diformat secara native di WhatsApp — code block, tabel, hyperlink, dan gambar ditampilkan dengan rapi.
